# node.js与element-ui实现简易后台搭建

# 技术栈

## 【前端】<br>
`vue-cli`:项目使用vue-cli脚手架搭建<br>
`vue-router`:项目路由搭建实现页面跳转交互<br>
`element-ui`:第三方UI库显示布局样式的快速搭建<br>
## 【后端端】<br>
`node.js`:使用node编写对应的RESTFUL api接口对数据进行操作<br>
`express`:node的框架使用<br>
`CORS`:使用CORS实现跨域<br>
